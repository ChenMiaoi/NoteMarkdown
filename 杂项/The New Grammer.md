# Grammer Type

--- 

> Obsidian give some extensions to modularity

- The general syntax is **"> [!TYPE]"**

## Info Type

> [!INFO]
> This is **INFO** Type

## Note Type

> [!note]
> This is **NOTE** Type

## Abstract Type

> [!abstract]
> This is **ABSTRACT** Type

- **But you can use the aliases key words: summary or tldr

### Summary

> [!summary]
> This is **ABSTRACT** Type

### Tldr

> [!tldr]
>  This is **ABSTRACT** Type

## Todo Type
