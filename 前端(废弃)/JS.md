## JS的三种输出

| 输出 | 作用 |
| --- | --- | 
| prompt | 输入框 | 
| alert | 弹出警示框，输出 |
| console.log | 控制台输出 | 

## JS的变量

### 声明变量

- JS中，使用var关键字来声明变量

```js
var age;
```

### 更新变量

> 一个变量被重新赋值后，它原有的值就会被覆盖，**变量值将以最后一次赋值为准**


## JS的数据类型

### 简单数据类型

| 数据类型 | 说明 | 默认值 |
| --- | --- | --- |
| Number | 数字型，包含整形和浮点型 | 0 |
